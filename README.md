# cpp-ftxui-tetris-assignment
C++大作业，AI+FTXUI实现终端俄罗斯方块
## 一、项目介绍
本项目为C++课程期末大作业，基于C++语言与FTXUI终端图形界面库开发，全程采用AI辅助驱动开发完成。
参考开源俄罗斯方块项目进行独立重构实现，未直接抄袭源代码。

## 二、实现功能
1.  经典俄罗斯方块完整核心玩法
2.  方块左右移动、顺时针旋转、加速下落
3.  满行自动消除、分数实时累计统计
4.  碰撞检测、游戏结束判定

## 三、开发环境
- 开发语言：C++17
- 图形库：FTXUI
- 构建工具：CMake 3.16
- 编译器：MinGW / MSVC

## 四、编译运行步骤
1. 下载本仓库代码
2. 在项目根目录下载 FTXUI 官方库，放在同级FTXUI文件夹
3. 编译运行：
cmake -S . -B build -G "MinGW Makefiles"
cmake --build build
.\build\tetris.exe
## 五、游戏操作
1.  ← → 方块左右移动
2. ↑ 方块顺时针旋转
3. ↓ 加速下落
4. 空格 一键落底
